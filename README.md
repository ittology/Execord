## ⚡ What is Execord?

Execord is an AI-powered Discord agent designed to eliminate the friction of traditional bot management. Instead of wrestling with complex slash commands, you simply describe what you want to happen.

> *"Create a community category with a general chat and 3 voice channels."*

Execord's AI interprets your intent, builds a sequence of actions, and executes them instantly via the Discord API.

## 💬 Try these prompts

| Goal | Example Prompt |
| :--- | :--- |
| **Server Setup** | *"Create a community category with a general chat and 3 voice channels."* |
| **Moderation** | *"Timeout @User for 1 hour for violating the rules."* |
| **Role Management** | *"Create a 'VIP' role in gold and give it to @User."* |
| **Bulk Cleanup** | *"Delete the 'Old-Archives' category and all channels inside it."* |
| **Visuals** | *"Generate a futuristic shield icon and set it as the server icon."* |
| **Permissions** | *"Create an announcements channel that only the @Admin role can write in."* |

## ✨ Core Features

- **Natural Language Control:** Speak to your server. No more memorizing `/` commands.
- **Bulk Operations:** Handle massive server changes in a single sentence.
- **Context Awareness:** Execord understands your server structure, channels, and roles by name.
- **AI-Powered Permissions:** Set complex permission overwrites without manual clicking.
- **Integrated Image Generation:** Generate and automatically apply server assets.

## 🧠 How it Works

Execord utilizes a sophisticated **Intent-to-Action pipeline**:

1.  **Parsing:** Your natural language prompt is analyzed by a Large Language Model (LLM).
2.  **Mapping:** The AI translates the request into a structured sequence of Discord API actions.
3.  **Verification:** Permissions are checked to ensure safety and compliance.
4.  **Execution:** Execord carries out the tasks sequentially, providing real-time feedback.

## 🚀 Getting Started

1.  **[Invite Execord](https://top.gg/bot/1369325346465517609/invite)** to your server.
2.  Ensure it has the `Administrator` permission (required for full automation).
3.  Type `/perform` followed by your request in any channel.
